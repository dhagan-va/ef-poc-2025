# Instructions On How to set up the project. 

- Install the following packages: 
	1. EdiFabric (For Readig and parsing edi files)
	2. EdiFabric.Templates.Hippa (For Readig and parsing edi files)
	3. Microsoft.EntityFrameworkCore (For DB interaction(ORM))
	4. Microsoft.EntityFrameworkCore.SqlServer (For interaction to a specific provider. (SLQ Server) )
	5. Microsoft.EntityFrameworkCore.Tools (For Code First Migrations)

- Register your EdiFrabric key in the pipeline. You can get it here [https://support.edifabric.com/hc/en-us/articles/360000280532-Free-code-to-master-your-EDI-files]
